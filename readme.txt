== Description ==
The plugin regularly removes translations from Divi Builder (Elegant Themes Divi & Extra themes & Divi Builder plugin). Removes only backend translations - Divi Builder. 

== Installation ==
1. Upload `pryc-wp-remove-languages-for-divi-builder` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
